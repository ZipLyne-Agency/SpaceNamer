# Security policy

Please report vulnerabilities privately through GitHub's security-advisory
feature for `ZipLyne-Agency/SpaceNamer`. Do not include secrets, private signing
material, or exploit details in a public issue.

Only the latest published SpaceNamer release is supported with security fixes.
Release artifacts are Developer-ID signed, notarized by Apple, and separately
authenticated by the Sparkle EdDSA key embedded in the app.
